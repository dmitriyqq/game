--------------- DESCRIPTION ------------------------------------------------
____________________________________________________________________________


Ceci est un modèle de vaisseau spatial. Il se presente en deux modes, un mode normal et un mode de combat. Les versions MAYA (mb et ma) contiennent des set driven key qui permettent de passer d'un mode à un autre sans changer de fichier. Pour les autres formats, des fichiers distinct pour chaque mode sont inclus. 

____________________________________________________________________________
--------------- Système requis ---------------------------------------------
____________________________________________________________________________


Ce produit a été créé avec Maya 2012 sur un PC. Il n'a pas été tester sur d'autres versions de Maya ou sur un systéme Mac.

____________________________________________________________________________
--------------- Installation Instructions ----------------------------------
____________________________________________________________________________


J'ai placé tout les fichiers dans le dossier racine, De ce fait, vous ne deveriez avoir aucun probléme pour charger les textures, sur n'importe quelle autre application 3D. Ouvrez les fichiers Maya (.ma ou .mb) ou importer le format de votre choix dans une scéne existante. Vous seriez, peut etre amenés à ajuster la taille du modèle.  

Pour les versions MAYA (ma et mb) j'ai configuré un systéme avec le set driven key afin de passer du mode normale au mode combat sans avoir à changer de fichier. Pour plus d'information, référez-vous aux tutoriels.
